## Instalación y funcionamiento en entorno de desarrollo

> nota: para entornos de producción deberá hacer el build de la aplicación web y el servidor correrlo con forever o pm2

***API***
```bash
cd preguntapi/server
npm install
npm run dev
```

***Aplicación web***
```bash
cd preguntapi
npm install
npm run dev
```
